## Setup

Clone the project

```bash
git clone https://github.com/taiogu/beststarter
```

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

```

## Development Server

Start the development server on `http://localhost:3000`

```bash
# yarn
yarn dev

# npm
npm run dev
```

## Production

Build the application for production:

```bash
# yarn
yarn build

# npm
npm run build
```
